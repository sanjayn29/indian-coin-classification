# 🪙 Indian Coin Classification

A deep learning project for classifying Indian coins using image classification techniques. The model is trained to recognize different Indian coin denominations from images.

## 🚀 Features

* Classifies Indian coins from images
* Supports multiple denominations:

  * ₹1
  * ₹2
  * ₹5
  * ₹10
  * ₹20
* Image preprocessing and data augmentation
* Transfer Learning using MobileNetV2
* Model evaluation with accuracy and loss graphs

* Python
* TensorFlow

## 📈 Model

* Transfer Learning using **MobileNetV2**
* Data Augmentation
* Image Normalization
* Softmax Output Layer
